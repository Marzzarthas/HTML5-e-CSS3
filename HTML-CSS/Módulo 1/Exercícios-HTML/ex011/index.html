Usando links externos

Você pode acessar um site de terceiros através de um link externo utilizando a tag <a>

Você pode acessar um site de terceiro abrindo em uma aba nova utilizando o parâmetro target="_blank" com um rel="external" dentro da tag a

Usando link internos

É um link que fará a relação com a nossa própria página, lembrando de utilizar a rel="next" dentro da tag a

Você também pode linkar páginas internas dentro de outras pastas, lembrando de selecionar a pasta com a página
