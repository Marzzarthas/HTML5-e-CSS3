Principais formatações

Negrito / Destaque

Nessa frase temos um termo en negrito usando a tag <b>(ela é uma tag não semântica, só altera a forma)

Nessa frase temos um termo em negrito utilizando a tag <strong> (que possui semântica pois dá destaque para a forma selecionada)

Itálico / Enfâse

Nessa frase temos umas frase em itálico utilizando a tag <i> (Obsoleto pelos mesmos motivos do negrito)

Nessa frase temos uma frase em enfâse (têm semântica) utilizando a tag <em>

Texto marcado

Podemos criar um texto marcado utilizando a tag <mark>

E no outro parágrafo temos outro texto marcado com outra cor no final, para uniformizar podemos utilizar a tag <style> na região head

Texto grande e pequeno

Aqui vamos criar um texto grande e um texto pequeno

Texto deletado

Podemos marcar um texto como excluído/deletado para indicar que apesar de lido, ele deve ser desconsiderado utilizando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois utilizando a tag <ins>

Texto sobrescrito

Para inserir por exemplo numeros elevados em equação x2+3y ; utilizando a tag <sup>

Texto subscrito

Para inserir textos subscritos para colocar números abaixo da linha, como por exemplo em formula molecular H2O ; utilizando a tag <sub>
